1、家庭心理学

老师：程晓玲
双语课程，考试开卷，但是老师讲得挺好，关于和父母，同学，恋人之间的关系。

2、大学语文

老师：陈建新
实话说上课非常无聊，完全是照着网上的资料在读，而且找的资料个人感觉也不佳，每次课要点名，不过反正去了的话也是可以安心自习的，无论文，要考试，考得挺细，不过好好复习就完全没问题，篇目的话课本上选，稍微有点多，内容以古文为主，对文言没什么兴趣就别选了，有兴趣的话还是建议选别的老师。

老师：许建平

那年是许建平老师第一次上这门通识课，老师是让我们自己选了几篇文章然后他逐篇讲解给我们听的，包括文章有很多词语的典故,经常是讲着讲着就会把涉及到的词语出处的那片文章也讲一遍，所以实际上一学期下来远远不止我们选的那几篇内容。

记得，我们全班的选择结果全部是歌颂爱情的文章（洛神赋一类）老师都无奈了•
考核是期中一篇用文言文写的自我介绍300字，期末是一篇用文言文写的读后感800字。写的时候很纠结，但是写完觉得好有成就感好有趣。

老师：周明初
我这个老师是交论文，不考试的。非常无聊。因为本学期同时在上“唐诗与经典研读”的通核，相比之下那门课就十分有料（当然也要多花很多时间，老师也很赞，讲了一个夏学期，最后一节课才匆匆把唐诗讲了一点。老师的口音有点重，时不时要跑偏。跑偏的内容比较偏激。如果是美分可能会产生共鸣。

3、汉字与东亚文化

老师：陈辉
推荐，老师上课讲的内容都挺有趣，主要是汉字对日本韩国的影响，老师课堂上鼓励做展示，而且一般做了展示老师就能把你记住了（小小地佩服一下老师地记忆力），期末要求6000字的和课程内容有关的论文，有时间可以好好写，全学期不点名，不过要是老师突然有兴致想要点一下你的名字也不是没可能（一个德语系的学姐被念叨了一个学期，还有个留学生），对课程内容有兴趣的话强烈推荐。

4、近代日本与东亚

老师：陈群元
台湾老师，每次进教室都捧着一杯奶茶，女神是林志玲，上课很有活力，讲得也很好，把近代以来中日各种历史事件略列了一遍，每周会有相关讨论题，也会布置阅读资料，长学期的3分的通识，一周三节课，两节老师讲一节同学展示，对日本文化和历史感兴趣的同学强烈推荐，期末写一篇3000字的课程论文，题目自选，期末也要考试，闭卷，这学期的题目是细述近代中国与东亚各地区的关系，就一题，给了五张白纸，有点小崩溃，不过因为之前就透露了，做好准备感觉还好。

5、法学基础

挺不错的课~就是课本没有什么用。由于是通识课，所以如果希望学系统的法律知识是不可能了，不过如果真是想了解了解法律方面的东西的话值得推荐~
记得当时的期末考核是一片当堂完成的案情称述还有一片论文（记得老师当时有很明确和严格的论文格式要求，这点很值得推荐~以后写论文都用得着而且规范格式是个很拉分的事~）

6、茶与健康

老师：何普明、屠幼英、王岳飞、吴媛媛
之一：茶学院开的课，知道茶学院不？很神奇的地方有没有？茶叶，好好玩的~

特别要提一下王岳飞老师，课上的很有趣很有趣，真的就是很有趣......总之不会亏的。

之二：据说是好课才选的，但是和我想象的完全不一样。我以为会教很多茶道、礼仪，结果一直在教心脏病、高血压、糖尿病，反正不是很喜欢。期末闭卷。

7、从神农本草到现代中药

老师：栾连军、陈柳蓉

期末一篇一片课程总结，不难吧！

8、中华礼仪及其重建

老师：关长龙
关于传统文化的内容，老师用8节课的时间主要讲了中国古代婚礼的概况，解读了《朱子家礼.婚礼》的内容，很多关于婚礼的内容很有趣也很好玩。期末是一篇论文，一个学期各种礼仪都涉及一点的节奏。老师很好，课也很好！

9、医学史

老师：余海
先是推荐所有学医的和打算学医的同学去听，能了解很多的东西~
期末是一篇和医学史相关的文章好简单吧！

10、中国传统历史文化 

老师：吴铮强

通识核心课程，不是按等级打分的，还是几年一次真的是好课。 分为理论课和讨论课两个课时一次课，老师理论课的内容讲的略慢，和讨论课脱节严重。记得那个时候的助教好厉害，我们都是小组PPT展示完了以后基本就是助教不断的讲相关的知识，相关的书，真的学到不少东西， 期末考试的时候是4选1的考题，题目事先知道然后就去考试就好了。之前还有一片读书报告，我那个时候好像是进考场写了篇小说就出来了，居然分数还不错。

11、无线网络应用

老师：金心宇、张昱

之一：

建议所有非计算机，信电有关的专业的同学去学。
缺点是一次有5个课时，有点花时间，可惜学分还是1.5。介绍一些互联网相关的事情，包括怎么设置IP，无线路由器或者是AP的设置。
有实验课~6个人3台电脑一组，无责任的表示只要组里有一个计算机系或者信电的同学抱大腿。
期末交每次实验的实验报告和期末大作业一份，平时就开始做，期末不费力。而且期末的大作业是有样例的，所以基本不存在不会的情况。

之二：
信电系老师开的通识课，技术类通识课，大多数是要写实验报告的。张老师是一个很严谨的人，也很负责。无线网络应用主要就是学一些计算机网络的基础理论，然后实验就做一些路由器、无限AP的各种模式的组网。学完这个课，至少玩路由器会玩了，一般的网络故障也不至于毫无头绪了，还是很实用的。我是一年级的时候学的，现在是否改革了不知道，但是相信只会越改越不水，但是内容丰富。关于做实验，只要按照课件做下来，都可以做出来的。晚上实验是你做完就可以走。做不完留到11点老师也会陪着你。不过没那么吓人，我和同学当时都是8点多一点就走了。提前预习一下课件，大概看看做点什么，其实很简单的。

16、音响技术与家庭影院 
老师：楼东武 
老师人超好，但是另一个崔老师么，我就不说什么了。课可以学到很多东西，包括像AA和cool editor软件的使用，成绩是按平时作业还有期末的一次测验决定的。还有就是这门课会去学校的录音棚要求自己给自己录一首歌，然后处理吗。

17、应用运筹学

老师：谈之弈
竺院数学建模的老师。当然应用运筹学没有数学建模上得那么难，主要还是了解算法的思想，方法，知道一点基础概念和基本操作方法就可以了。用老师的话说，很多东西后来就是交给计算机做的，人只要知道思想就可以了。当然具体的细节他也不会过多涉及的，因为毕竟人文社科类的同学也要学。

应用运筹学主要涉及：线性规划，组合优化，图论，博弈论四个方面。对于学计算机算法的人来说开拓一下思路也是有一点好处的，而且老师在讲算法的时候会有很多例子，他本人上课也是很风趣的，基本上只要认真听，就可以懂，而且不太会困，除非你一不小心脱节了再也跟不上了。我这学期给一个数学系的小伙伴推荐去上这个课，他去了，而且据说有一些数学系的大神也在上，总之就是课不算深，但是内容丰富，可以给理工科的同学开拓思路。
ps：谈神的大作业大多是写读后感之类的，读的是他提供的英文数学书或者期刊，其实谈神的资料非常好，可惜我都没有看完，一方面可以练英语，一方面那些数学趣味题是挺有意思的。比如什么亚瑟圆桌骑士问题等等。

18、课程名：进化的大脑与偶成的意识
老师：沈颖
沈老师是个挺憨厚的汉子,据说是医学生的解剖学老师之一,哪怕是理工科背景的同学，很多人上了大学就远离生物了吧!《进化的大脑与偶成的意识》是沈老师以前在国外的老板的一本书，其实本身就是一本很有趣的床头读物。上课主要是按照书上来讲，当然老师会提供更多生物学和医学的实验范例。建议大一的时候去学，因为那时候你的生物还没忘光~医学生去学就更加无压力的，但是感觉没什么必要呀，这个课就是简化版的神经科学。不过呢，对于外行同学，了解一下现代脑科学的进展，了解一下睡眠和梦境的科学解释、性与爱的脑科学起源，就当听讲座也不错。只要上课听了，考试都很简单的。
19、数字电视基础与检测
老师：陈鹏飞
长学期的通识课，隔周一次实验，写了7份实验报告，自己给自己出一张期末考卷，最后还要考试，考试就是用个万用表去判断电路板哪里故障了导致电视坏了，比如公共通道某个电阻断路了导致屏幕出现雪花等等。其实现在电视已经不是这样老古董了，这个课的真正意义在于给理工科的同学培养一种查找电路故障能力。尤其是要学模电实验的同学。模电实验自己的板子故障在哪里找不到很痛苦吧？电视机电路板比你的板子复杂一百倍！但是在老师的指导下，很快你就能找到门路，学会看电路图，电路板看不懂都不要紧，故障你能找出来就好了。

这个课是第一次让大家认识到电路不是你电路原理模电里面学的那几个简单元件的组合，你会算基尔霍夫方程组都不一定找的到哪里有故障，相信会让大家有学以致用的快感。
这门课前半个学期主要让大家玩最新型的智能电视，还有学习使用premiere进行简单的视频剪辑。总之内容是很丰富的~

20、数学与人类文明
老师：程晓良
就教学内容来说很完善很系统，听一遍下来外加看蔡老师的书，至于教学我觉得只能是中规中矩的吧，会要求写一篇论文，后几周会给时间展示，只要和数学相关的课程就好了，如果说真的要从这门课中学到什么该是自己努力吧，课程本身没有太多的亮点，不过也不麻烦。
21、化学与人类文明
老师：谢玉群
也就是现在的科学与生活，讲的内容还算有些亮点吧，因为存在一些尺度的部分，不过其实也仁者见仁智者见智的事情，总体来说比单纯的文明类的课程多了一些野地里的东西，非常规的知识，也就听听还是可以的吧。闭卷考试和那本书比较有关系，大致除了很个别的不容易找到，基本觉得还行吧！
22、生命科学与生物技术导论实验

老师：吴敏等
实验课，4个整天，会有各种植物动物微生物等，有好玩的也有枯燥的，可以去精品课程的网站上去看具体实验内容，实验报告只需要填写实验册就好，当场就搞定也不太麻烦。我觉得选这么课程主要在于兴趣吧，的确会有所获得实验产品，时间上1学分的课程还是比较费时的。
23、中国古典文学欣赏
老师：孙敏强
我比较推荐听一下这门课程，不过就一次足矣，因为他开的专业课据说和这门通识课大体上也差别并不太大，而且这是一个很具有感性的老师，从知识上来说可能系统性较差一些，但是感受却能很深刻。讲课内容基本是从他写的诗意诗心的顺序下来，这本书我觉得写得很不错！每节课都会再给我们看一些他自己的作品，对于心灵的慰藉与感悟真的很赞。要从心体会下去，因为大多是从他的感悟出发。于是基本处于欣赏的角度，可能理论性的稍微欠缺一些吧！但是，总觉得感悟一回还是值得的。另外这老师待人接物蛮好，据说很多同学都会私下里和他联系交流帮忙什么的。
24、现代遗传学概论
老师：石春海
感觉大致内容高中生物基本就覆盖上了，老师讲课有时候会有风趣的地方稍微可以调节一下气氛，不过似乎感觉有点冷笑话的味道。每节课都需要点名所以必须到恩，还会偶尔抽到点名，反正过去写作业也是可以的。最后论文就这样搞定了，只能说是一般的课程，也不太麻烦。
25、当代中国经济
老师：徐晓虹
这个老师的上课很敬业！感觉准备的很充分，内容安排也很合理，讲得也很不错，学到的东西也很多，还会补充额外的阅读资料，中英文的都有。当然会觉得比较麻烦，因为要交论文、读书报告并且有开卷考试。
26、法学基础
老师：梁清华
这个老师讲课很不错相信大家都知道！当然小组讨论和各种读书等要求也是很多的，虽然可能有点思修的法律那块的味道，但是感觉这课程教授的案例等等还是蛮不错的，特别是讨论。
27、中国现当代诗歌赏析
老师：吴晓
就知识理论体系来说，这门课的结构化很清晰，能够对于现代诗歌的各方面赏析有很理性的分析和讨论，比如意象等，但是总觉得略缺一些人文关怀的情绪感吧！期中写一首诗歌，期末一份论文 。
28、管理心理学
老师：陈大柔
上课很有趣，老师的确蛮幽默，讲课也很有意思，而且基本上课上都会有一些心理测试的实验 。
没有什么太大问题，似乎也很多人蛮喜欢的。

30、经济学基础

与其上这么个通识不如去微观宏观的大类课都去上一下来得美好一些。8次课完全就讲不了多少也讲不清楚多少东西，学到的也很少，不推荐 。
老师：陈菲琼

31、性科学与健康 
老师：许毅
由附属医院这个科室的几个老师轮流上课，只能说这课程开始的几节课感觉上还不错，不过感觉是越上越无趣，特别是最后的几个老师就略无聊很多了。当然因为涉及的尺度和内容还是有人闻名去听课，但是由于在华家池所以也很麻烦的 。
32、生活中的微生物发酵实验

33、宋词研究
老师：陶然
开在西溪，闻名许久，的确是一门很好的课程！老师很帅的，而且学到的东西也的确很多，不过要是能更好就好了，大概是因为之前听说以后期望太高的缘故，总觉得可以更好一些。

34、英语短篇小说欣赏
老师：郭国良
长学期的通识，一个学期要读6篇短篇小说（short story）。课是在紫金港东7的语言教室上的，所以一个班的人数并不多。老师是很好的一个人，上课讲的很详细，会把小说的细节都剖析出来。不过这个学期学生好像比较冷场，大概是有一半左右的学生是留学生的缘故吧。期中非留学生要做一份翻译作业，期末考试是开卷，可以带除了电脑之外的一切东西。对于那些对英语文学比较感兴趣的同学还是很合适的。
35、书画鉴赏及艺术市场
老师：黄鼎
老师超赞，非常专业！对书画感兴趣的同学值得去上。貌似从未点名。作业包含一次去美术馆、博物馆参观，回来写一篇文章；期末随堂鉴赏书画，给分也不错 。
36、电工电子工程训练
老师：汤巍松
课程是打等级的。我之前都不知道，不过貌似很多实验类课程都是打等级的。不过课程内容还是蛮好玩的。对于一个不用焊电路的文科生来说也算是一种补充。不过和所有技术类通识一样，对电路无感的文科孩子慎选。
37、机电一体化
老师：祝长生
课程无点名，只上七节大课，只是要记笔记，期末半开卷只能带笔记，考的基本上是笔记的内容（也就是PPT的内容）。老师不让拷PPT的说，课程内容有难度的。
38、生命科学（通识核心课程）
老师：吴敏，唐建军
这两个老师是我大学见过最喜感的两个老师~上课很有意思，内容都是生物学科的基础和科技前沿。经常点名，不要翘课也不要迟到哦，不过老师讲得这么好就不要翘课啦。讨论课略显沉闷，不过积极发言绝对有好处的~要交一篇论文和闭卷考试，客观题只有靠积累了，生科的童鞋完爆。不过考试内容老师上课都会讲到，只要认真听就没有压力啦。给分还不错的。
39、生态安全的系统分析
老师：杨京平
新生研讨课，但说实话上了之后觉得很失望，当时也没有退课，所以也就这么水下来了。这个老师给我的感觉是他不想上课，上课就照PPT读一遍再放个两节课的纪录片，讨论课的话题也很空泛。所以觉得学到的东西太少。想真正学些东西的孩子慎重。 期末一篇三千字论文即可。
40、心理学及应用
老师：何贵兵
老师上课风格可能略沉闷，因为我看周围人好好听的没几个。。。但我因为初次接触心理学课程并且对心理学很感兴趣所以也听得很开心。课程考核是三次小作业，期末开卷卷考试，两道主观题。
41、动物基因与人类发展
老师：徐豫松
老徐是个有喜感的人，很有商业头脑feel，无时无刻不在想着怎么把科技成果转化为商业价值，常提起的就是玛丽莲梦露，讨论课多到无法忍受的地步，注重学生表达能力与创造性的培养热情，会为学生着想。最后一次课是复习课，把复习课的ppt一张张拍下来，再打印出来带上考场照着一字不落地抄就行。但是，我上了这门课之后，才发现我并没有像想象中那么喜欢生物，尤其是基因这块。讨论课至少两次发言，期末开卷4道主观题 。
42、人生美学
老师：张应杭
大学第一门通识，超级赞的课！老师上课会讲很多类似心灵鸡汤一样的故事和话语，但是一点也不觉得突兀或者做作，每次上完课都是受益匪浅。我上课的时候由于老师研究生的课和这个课时间冲突，8次课4次讲课4次看电影，一篇电影随感，期末是开卷考。
43、插花艺术与盆景
老师：李方
很好的课，很气质的老师！4次实践课都是插花，2－3人一组，选一次评分作为平时成绩。当时老师让我们自己选考试或者论文，最后有同学说考试周想回家于是就论文了，总之挺欢乐的。
44、中英语言文化比较
老师：吴越民
选课的时候写着非双语教学，结果居然是英文教学！长学期2节课1.5分的，从学时来说是不划算的，但是可以长长见识什么的。期末是一篇论文以及闭卷考，论文可以中文可以英文，英文分数高一些，考试的内容基本都在ppt里，英文考试哦。而且长学期的课不是你每次都听就可以掌握的，学到后来前面的就完全忘记了，总之是还算不错的课。
45、.管理沟通
老师：陈随军
管理学院特色之姓名牌和各种小组讨论活动。不过最想吐槽的还是陈老师的出分速度啊，秋学期的课，而且是最后一节课当堂考，居然到冬学期考试周结束后好几天才出，冬学期的通识都先出分了。
46、跨文化沟通
老师：程晓玲
又是长学期2节课1.5分的通识，竺可桢学院开课，基本每节课都会签到，就是一张名单传来传去签名字。另外还有小组presentation，期末是开卷考。由于我学了很多cross-culture的课了，所以觉得很简单。
47、基因与遗传突变A
老师：韩凝
这门课都是基础遗传学基础的讲授，如果不喜欢生物学的真的不建议学。关于得分情况。首先，每次课堂测验都要去，虽然不知道哪一次韩老师会点名，最后是最后一节课做一次测验，占50分，主要是主观表述，给分不错的，也能学到不少知识。
48、海洋生物学
老师：杨万喜
有关海洋生物的通识课，没有课本，而且PPT也不是很系统。有两次考试，很无厘头，有一百道选择题，有一部分还是和课程内容相关，感觉很累，而且还是以记忆背诵为主。老师很好，讲课也很认真，但是有照着念的嫌疑。比较尽责，传播一些正能量，但是还是建议大家不要选。
49、生物工程导论
老师：吴绵斌
内容比较多，课上不停讲。有一次小组展示，点了一次名，貌似还记录了来了但没好好听课的，考试很简单。
50、电工电子学概论
老师：应群民
老师挺专业的，但是真的不太听得懂！！！考试前最好把以前卷子的计算题搞会。

51、亚洲经济发展
老师：陈建军
老师上课很精彩，主要讲讲日本、韩国、台湾的经济，附带讲讲历史、名族性格，日本内容居多，不点名，有小组展示（所有组都做，选部分最后一课讲），无考试，写一篇1400论文，选题很重要！强力推荐！
